# social-media-dashboard
A single page application using react, dva and ant-design

## Installation

Clone project:
```bash
git clone git@github.com:locronaldo/social-media-dashboard.git
```

Install dependencies: 
```bash
npm i 
// or 
yarn install
```

Run:
```bash
npm start
// or 
yarn start
// then open: `http://localhost:8000`
```

Test:
```bash
npm run test
```
